## Jot It Down
A simple react app to make notes, add images, save links and have to do lists.

## Functional Requirements:
- Add/Edit/Delete/View notes
- Add/Edit/Delete/View images
- Add/Edit/Delete/View hyperlinks
- Add/Edit/Delete/View lists
- Authenticate User

## Non Functional Requirements
- Forget password
- Added features in Notes
    - Add more notes
    - Add Code Snippets
    - Add images
    - Add tables
- Formatted text
